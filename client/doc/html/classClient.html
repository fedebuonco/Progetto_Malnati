Detailed Description

The main class of the client.

It represent an object that will connect to the server specified in the constructor and will ask to a watcher to watch a specified folder.

bool Client::Auth ( )

Will send username and password.

After that it will shutdown the send part of the socket thus providing the server a way to tell that the connection is over. After the authentication the socket is basically useless and needs to be shutdown completely.

Returns
'True' if Auth went ok and user has successfully logged in, 'False' if not.
