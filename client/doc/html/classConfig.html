◆ SetDefaultConfig()

void Config::SetDefaultConfig ( )

It resets all the fields inside configuration file (config.json) with default value:

  • Default Endpoint: 127.0.0.1 3333
  • Username and password: empty
  • Folder: empty

    It also reset the configuration structure (create the correct folder and file) if it is broken for some reason (i.e. file or folder doesn't exist).

This function must be used when the program detect problem during configuration phase. In this way we restore all with the default files.
