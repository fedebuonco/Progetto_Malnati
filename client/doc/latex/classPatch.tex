\doxysubsection{Detailed Description}
Wraps in a compact way all the differences between the client and server. This difference is computed starting from the \mbox{\hyperlink{classTreeT}{TreeT}} of the client and of the server. Will be \char`\"{}applied\char`\"{} to the server.
