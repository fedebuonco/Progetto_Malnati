Detailed Description

Represents an high level acceptor.

Accepts and instantiate a Service. When constructed it instantiates an acceptor socket and starts listening on it.

Creates an active socket and accepts an incoming connection by the client thus linking them and creating a Service class that will take care of the handling.

This service class will obtain the pointer to the socket here created. The accept is blocking and has a own list of incoming connections.
